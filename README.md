# Hexagon Strategy Game

Повнофункціональна браузерна стратегічна гра захоплення територій.

## Механіки ГРИ
- **Захоплення:** Атакуйте сусідні території, щоб розширити свою імперію.
- **Війська:** Кількість військ визначає силу атаки та захисту.
- **Підкріплення:** Кожен хід ви отримуєте нові війська залежно від кількості ваших територій.
- **ШІ:** Комп'ютерний опонент, який намагається захопити карту.

## Як грати
1. **Фаза підкріплення:** Клікніть на свою територію (синього кольору), щоб додати війська.
2. **Фаза атаки:**
   - Клікніть на свою територію з >1 військом.
   - Клікніть на сусідню ворожу (червону) або нейтральну (сіру) територію.
3. **Завершення ходу:** Натисніть кнопку "Завершити хід", щоб передати чергу ШІ.

## Технічні деталі
- Побудовано на Vanilla JS та HTML5 Canvas.
- Використовує систему координат Axial Hex Grid.
- Модульна архітектура (Engine, UI, AI).

## Запуск
Просто відкрийте `index.html` у будь-якому сучасному браузері.
